Description

During the semester of January 2023-April 2023 I learned the fundamentals of UX/UI design. Throughout the design process I was challenged to create a solution for a problem of my choice. In this case it was to create a website/app that would allow users to freely browse and book lawn care providers. This solved the issue of hunting down Contractors and making un-needed phone calls.

Design Process and Documentation

To kickstart this project we used a variety of methods in order to gather user input and make a well though out design. This information was placed in a documented folder and became our requirements elicitation. In order to find certain Must, Should, and Could have features we needed to hold interviews. These interviews gave us many User Need statements that we could build on and it allowed us to imagine what to create
